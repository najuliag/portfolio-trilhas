Sobre mim

Sou acadêmica de Engenharia da Computação na UEMA e uma desenvolvedora apaixonada por tecnologia desde os 16 anos. Tenho experiência em Java e Spring Boot para criar aplicações web, juntamente com banco de dados (MySQL, MariaDB e PostgreSQL). Utilizo HTML e CSS para criar interfaces intuitivas e responsivas e trabalho com Docker para containerização de aplicações, garantindo portabilidade e eficiência. Estou sempre atualizada com as últimas tendências e busco desenvolver soluções inovadoras e eficientes.
